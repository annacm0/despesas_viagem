# Cálculo de Despesas de Viagem

Este é um programa desenvolvido em **C** que ajuda a estimar o custo de combustível para uma viagem. O código foi criado como um exercício prático de lógica e operadores matemáticos.

## Como funciona
O programa solicita dois dados principais:
1. A **distância** total da viagem (em quilômetros).
2. O **preço** do litro do combustível.

O cálculo baseia-se em um consumo médio fixo de **10 km por litro**. A fórmula utilizada é:

$$Custo = \left( \frac{distância}{10} \right) \times preço$$

O resultado final é exibido formatado em Reais (R$).
